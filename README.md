# 3D Software Rasterizer
A software-based 3D rasterizer engine made from scratch with C and SDL. Made with the guidance of Pikuma's "3D Graphics from Scratch" course. It parses data from an OBJ file for vertex data and a PNG file for texture data. It can render both quad and triangle meshes. Some other key features include:

*View frustum culling using Sutherland-Hodgman's algorithm.

*Bresenham's algorithm for faster lines.

*Backface culling

*The following render modes: Wireframe, textured wireframe, textured mesh and blank mesh with flat shading illumination.

*A Doom-style camera that can be controlled with the keyboard's directional keys.
